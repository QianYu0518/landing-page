# Company Landing Page

A responsive company landing page built with HTML, CSS, and vanilla JavaScript.

## Features
- Mobile-first responsive layout
- Sticky navigation and mobile menu
- Hero, services, projects, about, CTA, and footer sections
- Modern CSS grid/flexbox layout
- Accessible headings and navigation controls

## Run
Open `index.html` in a browser or use the VS Code Live Server extension.

## Portfolio customization
Replace the company name, email, services, and sample projects with your own content.
